Edit Pinned Settings

    iOS Settings Prototype

    This prototype simulates the iOS Settings interface, allowing users to interact with various settings and observe how they affect the device's status and appearance.

    Features:

    • Dynamic Search Bar: Search for settings by name or keywords. Results are highlighted and can be directly interacted with (e.g., toggling switches or adjusting sliders).
    • Pinned Settings: Quick access to frequently used settings. Long-press to edit which settings are pinned.
    • Collapsible Sections: Organize settings into categories like "Recently Used", "Display & Appearance", "Connections", and "System". Click headers to collapse/expand.
    • Interactive Settings (in Search Bar):
      • Toggle switches for settings like Wi-Fi, Bluetooth, Airplane Mode, etc.
      • Brightness slider to adjust screen brightness.
      • Text size adjustment (in search results).
